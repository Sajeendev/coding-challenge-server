## Installation

```bash
# makeusre you have installed nest cli, otherwise install it
npm i -g @nestjs/cli


# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
